# Dungeon (Zork)

Ce projet est un humble hommage au premier jeu informatique que j'ai utilisé en 1986 sur un mainframe Burroughs 6700 et que j'ai voulu reprendre en français tout en apprenant les rudiments du langage Python.

Dungeon a été créé à la fin des années 70 par quelques potes du MIT. Il a été à la source de la trilogie Zork I, II et III développé par Infocom. D'autres info [ici](https://en.wikipedia.org/wiki/Zork).

## Introduction originale


Welcome to Dungeon!

   Dungeon is a game of adventure, danger, and low cunning.  In it
you will explore some of the most amazing territory ever seen by mortal
man.  Hardened adventurers have run screaming from the terrors contained
within.

   In Dungeon, the intrepid explorer delves into the forgotten secrets
of a lost labyrinth deep in the bowels of the earth, searching for
vast treasures long hidden from prying eyes, treasures guarded by
fearsome monsters and diabolical traps!

   No DECsystem should be without one!

## Démarrage

Le jeu se démarre dans un terminal avec :

```bash
python objects.py
```

## Commandes

### Déplacements
nord, est, sud, ouest,  
nord-est, sud-est, sud-ouest, nord-ouest  
monter, descendre, entrer, sortir

### Actions
prendre, poser, mettre _objet_ dans, poser _objet_ sur,  
ouvrir, déplacer, allumer

### Infos
regarder : infos sur l'endroit où l'on se trouve  
examiner : un objet  
aide : les commandes possibles  
inventaire : ce que vous portez sur vous   

### Quitter Dungeon
quit, ou exit
